# LazeEditor

LazeEditorは、Monaco Editor ライブラリを使用して開発されたウェブベースのコードエディターです。Laze言語に特化した高品質な開発環境を提供します。

## 機能

- Laze言語の構文ハイライト
- カスタマイズ可能なエディターテーマ（括弧ハイライト対応）
- 自動補完とコード整形
- Monaco Editorライブラリとの統合による高品質なコーディング体験

## 必要環境

- Node.js
- npm

## 使い方

1. リポジトリをクローンする:
   ```
   git clone https://github.com/your-username/LazeEditor.git
   ```

2. 依存関係をインストールする:
   ```
   cd LazeEditor
   npm install
   ```

3. プロジェクトをビルドする:
   ```
   npm run build
   ```

4. 開発サーバーを起動する:
   ```
   npm start
   ```

   LazeEditorは `http://localhost:8080/` で利用可能です。
