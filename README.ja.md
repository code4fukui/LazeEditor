# LazeEditor

LazeEditorはモナコエディタ (Monaco Editor) を使用して開発された、Laze言語に対応したコードエディターです。

## 機能

- Lazeプログラミング言語の構文ハイライト
- Lazeコードの自動補完
- カスタマイズ可能なエディターテーマ（括弧ハイライト対応）
- コード整形

## 必要環境

- Node.js
- npm

## 使い方

1. リポジトリをクローンする
2. `npm install` でパッケージをインストールする  
3. `npm run build` でプロジェクトをビルドする
4. `dist/js/editor.js` をブラウザで開く
