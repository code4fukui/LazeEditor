# LazeEditor

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

The LazeEditor is a web-based code editor that uses the Monaco Editor library to provide a rich coding experience for the Laze programming language.

## Features

- Syntax highlighting for the Laze language
- Customizable editor theme with support for brackets highlighting
- Auto-completion and code formatting
- Integration with the Monaco Editor library for a high-quality coding experience

## Requirements

- Node.js and npm installed on your system

## Usage

1. Clone the repository:
   ```
   git clone https://github.com/your-username/LazeEditor.git
   ```

2. Install the dependencies:
   ```
   cd LazeEditor
   npm install
   ```

3. Build the project:
   ```
   npm run build
   ```

4. Start the development server:
   ```
   npm start
   ```

   The LazeEditor will be available at `http://localhost:8080/`.
